Unmanned Aerial Vehicle Patrol System Project

An unmanned aerial vehicle (UAV), commonly known as a drone, is an aircraft without a human pilot aboard. UAVs are a component of an unmanned aircraft system (UAS), which include a UAV, a ground-based controller, and a system of communications between the two. The flight of UAVs may operate with various degrees of autonomy, either under remote control by a human operator or autonomously by onboard computers.

The team will design and implement a webpage system that can communicate and control the basic control of an aerial vehicles, including take off, landing, position control through SDK provided by the aerial vehicle provider. The planned UAV used in this project is DJI M200 onboard SDK.

It will be able to do machine learning based automatic object recognition module from videos taken by an UAV onboard camera, and there is a system can automatically track a single object (extra bonus) from UAV onboard video stream.


Intelligent Tracking

The automatic object recognition module will search the target, which is based on machine learning. And then, automatically track it from UAV onboard video stream.

Smart Routing

Smart routing is also available in patrol system, CPU will calculate a optimal route for each vehicle, in order to ensure the maximum efficiency.

Auto Self-checking

The System is able to do self-checking, which including runing status, flying attitudes and battery remaining, etc. UAV will come back automatically before the battery life running out.
